# 🌐 Personal Portfolio Website

## 📌 Overview

This repository contains my personal portfolio website. It is designed to showcase my profile, technical skills, and academic background as a Computer Science student. The website acts as a digital platform where visitors can learn more about me and explore my work and interests in technology.

## 🎯 Objective

The main objective of this portfolio website is to create a professional online presence and present my learning journey in the field of Computer Science and Artificial Intelligence.

## 🛠 Technologies Used

The website was developed using the following technologies:

* **HTML** – Structure of the website
* **CSS** – Styling and layout design
* **JavaScript** – Basic interactivity
* **Git & GitHub** – Version control and hosting
* **Visual Studio Code** – Development environment

## 📂 Website Sections

The portfolio website includes the following sections:

* **Home** – Introduction and welcome section
* **About Me** – Information about my background and interests
* **Skills** – Technologies and tools I am learning
* **Projects** – Overview of my academic and personal projects
* **Contact** – Information to connect with me

## ✨ Features

* Simple and modern design
* Easy navigation across sections
* Responsive layout for different devices
* Structured presentation of profile information

## 🚀 Purpose

This portfolio website serves as a platform to present my skills, interests, and growth in technology while demonstrating my ability to build and manage a web-based project.
